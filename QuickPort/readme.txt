==========
== KEYS ==
==========

w, s, a, d................move (run)
left shift................walk
space.....................jump
l.........................turn your light on/off
b.........................stop/start music
m.........................next music track
n.........................previous music track
esc.......................exit

==============
== GAME.CFG ==
==============

[RESX] 		- X resolution
[RESY] 		- Y resolution
[COLORDEPTH] 	- Depth of the color buffer
[ZBUFFERDEPTH] 	- Depth of the z buffer
[TEXTURESIZE]	- Size of textures to use (not implemented yet, keep at 256)
[NICEGRAPHICS]	- Sets some general options for OpenGL to opt to draw for quality or speed (may only effect older machines)
[MIPMAP]	- Mip mapping the textures will make the game look better and run faster
[FILTERING]	- Bilinear filtering will make textures appear smooth instead of blocky
[FULLSCREEN]	- Fullscreen mode or windowed
